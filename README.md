# HighlightEngine

## First of all install OpenAL and update OpenGL drivers so your video card could support GLSL shaders version 400.

## This is a game engine which is written using C# and OpenGL with GLSL shaders.

### Engine features:
* # Lighting
* Point light;
* Directional light;
* Phong shading;
* Normal mapping;
* Shadow mapping;
* # Postprocessing
* Light shafts;
* Lens flares;
* DoF;
* Bloom;
* # Distortions
* Water planes;
