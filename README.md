# HighlightEngine

## First of all update OpenGL drivers so your video card could support OpenGL 4.0.

## This is a game engine which is written using C# and OpenGL with GLSL shaders.

### Engine features:
* # Lighting
* Point light;
* Directional light;
* Phong shading;
* Normal mapping;
* Shadow mapping;
* # Postprocessing
* Light shafts;
* Lens flares;
* DoF;
* Bloom;
* # Distortions
* Water planes;
*# Animation
